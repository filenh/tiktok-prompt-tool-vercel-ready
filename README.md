# TikTok Prompt Generator — Vercel Ready

Đây là bản React + Vite deploy-ready. Bạn có thể upload lên GitHub rồi deploy bằng Vercel.

## Chạy thử trên máy

```bash
npm install
npm run dev
```

## Deploy lên Vercel

1. Tạo repo mới trên GitHub.
2. Upload toàn bộ folder này lên repo.
3. Vào Vercel → Add New → Project.
4. Chọn repo.
5. Giữ mặc định:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
6. Bấm Deploy.

Sau khi deploy, Vercel sẽ tạo link để bạn gửi cho người nhà dùng.

## Update tool

Sau này sửa code rồi push lên GitHub, Vercel sẽ tự deploy lại.
